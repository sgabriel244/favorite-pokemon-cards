The Jungle Set is the second set released for the Pokémon Trading Card Game back in 1999 (in English). This set continues to feature Pokémon from the Generation I. It is a themed set featuring Pokémon that you can encounter at the Safari Zone in the Red, Green and Blue Pokémon games.

The main Pokémon shown on the pack images above are Flareon, Wigglytuff and Scyther, which can be encountered at the Kanto Safari Zone.

Pack release date:
Japanese: March 5, 1997
English: June 16, 1999
