Get back here!

Base Set 2 is a reprint of the Base Set and Jungle Set for English players. The Fossil Set was the last one to use the Galaxy Star Holofoil and now the Cosmos Holofoil is being used. Some of the cards were reprinted and some weren't.

The Pokémon included as the cover of the booster packs are Mewtwo, Raichu, Pidgeot and Gyarados.

Pack release date:
English: February 24, 2000
